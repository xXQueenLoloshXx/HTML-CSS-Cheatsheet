HTML Form Reference

Form Tags

Tage Name Description
<form> From an element that contains one or more of form elemetns
<input> Input and element that can be be displayed in several ways
<label> Label an elementthat defines a label for several form elements.

Input Type Attributes

Value Description
"text" defines a single-line text input field
"password" defines a password field
"radio" defines a radio button
"button" defines a button
"color" is used for input fields that should contain a color.
